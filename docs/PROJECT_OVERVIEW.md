## 2. 项目背景

环境试验箱用于模拟温度、湿度、低气压、温度冲击等环境条件，广泛应用于产品测试和可靠性验证。现场设备通常自带本地触摸屏控制功能，但实际使用中仍然需要一套**上位机软件**来完成：

- 统一管理多台试验箱
- 实时查看设备状态和关键参数
- 远程下发控制命令
- 编辑并下载程序曲线
- 保存历史数据并进行回看分析
- 查看报警记录和运行日志
- 支持不同品牌 PLC 与不同型号试验箱接入

本项目开发一套新的上位机软件，**替代或升级现有系统**，使其更稳定、更易维护、更适合后续扩展。

## 5. 五条设计原则

1. **配置优先** —— 不同设备/PLC/通道数量，通过配置文件和协议模板适配，不改主程序代码
2. **模块分离** —— 界面、通信、数据存储、报警处理、日志记录分开设计，互不干扰
3. **单设备隔离** —— 某一台设备通信异常，不影响其他设备正常运行
4. **操作安全** —— 所有控制命令先检查设备状态（在线? 远程? 范围? 当前状态允许?），再决定是否下发
5. **便于维护** —— 目录结构清晰，日志清晰，配置可备份，便于交接和后期维护
